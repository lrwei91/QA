# Account Access Domain

## 1. Covered modules

- `auth-center`
- `admin-user`
- `role-permission`
- `organization`
- `kyc-certification`

## 2. Common business objects

- 管理员账号
- 登录凭证、验证码、token、session
- 角色、菜单、按钮、接口权限
- 数据范围、组织树、组织归属
- KYC 认证、人脸验证、CPF 验证、税收报表、IRPF

## 3. High-risk test concerns

### 3.1 客户端

- 登录页、列表页、详情页、弹窗是否正确展示
- 表单校验、错误文案、按钮置灰、可见性是否正确
- 菜单、按钮、数据范围是否按权限呈现
- 组织树、搜索、筛选、分页是否正确回显
- 认证列表页、人脸验证页、CPF 输入框、报表展示、筛选条件

### 3.2 账服

- 登录鉴权、会话续期、失效后行为、错误次数限制
- 账号启停、重置密码、绑定角色、绑定组织的写库与状态同步
- 越权访问拦截、数据范围过滤、缓存刷新、权限生效时机
- 关键操作是否写入审计日志
- KYC 接口对接、人脸验证服务、CPF 校验、税收计算、报表聚合

## 4. Common regression expansion rules

- 账号启停或密码策略变化 -> 补查 `auth-center`、`operation-log`
- 角色权限变化 -> 补查所有被授权业务模块与 `operation-log`
- 组织范围变化 -> 补查 `dashboard` 统计口径和 `admin-user` 数据回显
- 登录与会话需求 -> 必查权限菜单、失效跳转、日志留痕
- KYC 认证规则变化 -> 补查 `player-management`、`finance-system`、`risk-management`、`report-system`

## 5. Typical case ideas

### 认证中心

- 正常登录、错误密码、验证码失效、会话过期、重复登录、退出登录
- 客户端关注登录页展示和错误提示
- 账服关注鉴权逻辑、token/session 生效与失效、风控限制

### 后台账号管理

- 新增账号、编辑账号、启停账号、重置密码、导入导出
- 客户端关注列表与弹窗交互、筛选回显
- 账服关注字段校验、重复账号校验、状态变更、通知与日志

### 角色权限

- 菜单权限、按钮权限、接口权限、数据范围
- 客户端关注菜单/按钮可见性和置灰逻辑
- 账服关注接口越权拦截与数据范围过滤

### 组织架构

- 组织新增编辑、层级关系、组织停用、组织归属变更
- 客户端关注树结构展示和节点操作
- 账服关注停用限制、层级合法性、归属同步

### 认证内容 (KYC)

- 数据分析报表、出入款报表、投注报告、供应商表现
- 最新存款、停机时间、无存款用户、首次存款
- KYC 认证记录、人脸流程统计、CPF&面部验证
- 玩家税收发送记录、大额投注报表、大额奖金报表、个人所得税 (IRPF)
- 客户端关注认证列表、人脸验证页、CPF 输入、报表展示
- 账服关注 KYC 接口、人脸验证、CPF 校验、税收计算、报表聚合
