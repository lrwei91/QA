# Operation Management Domain

## 1. Covered modules

- `frontend-management`
- `game-management`
- `operation-management`
- `affiliate-management`
- `partner-agent`
- `channel-management`
- `vendor-activities`

## 2. Common business objects

- 大厅分类、自定义皮肤、下载页、APK 包、马甲包、三方登录、多货币
- 游戏配置、厂家列表、游戏点数、外部游戏、体育游戏、上下分订单
- 运营活动、公告、广播、多语言文本、站点地图、跨服中台、机器人客服
- 全民代、合伙人、代理、佣金、团队层级、ROI、排行榜
- 渠道列表、渠道类型、渠道数据、Telegram 渠道、新玩家跟踪
- 厂商活动（FC/FG/TADA/YELLOWBAT）、锦标赛、免费旋转、抽红包、礼包码
- 充值、提现、代付、转账、出款、通道设置
- 玩家列表、分层、VIP、标签、回归玩家、账号注销
- 游戏记录、金币变化、充提记录汇总

## 4. Common regression expansion rules

- 前端资源配置变化 -> 补查 `game-management`、`operation-management`、`operation-log`
- 游戏配置或点数变化 -> 补查 `game-golden-flow`、`report-system`、`finance-system`
- 运营活动配置变化 -> 补查 `notification-center`、`report-system`、`operation-log`
- 佣金规则或团队关系变化 -> 补查 `finance-system`、`affiliate-management`、`partner-agent`
- 渠道数据变化 -> 补查 `report-system`、`affiliate-management`
- 厂商活动奖励变化 -> 补查 `finance-system`、`player-management`、`report-system`
- 充值提现规则变化 -> 补查 `risk-management`、`report-system`、`player-management`
- 玩家分层或 VIP 规则变化 -> 补查 `report-system`、`finance-system`、`operation-management`

## 5. Typical case ideas

### 前端管理

- 大厅分类配置、皮肤切换、APK 上传、包版本管理、马甲包配置
- 三方登录配置、多货币切换、合作方配置、社媒配置
- 客户端关注配置页交互、预览效果、上传进度、版本展示
- 账服关注配置写入、文件存储、版本控制、缓存生效

### 游戏管理

- 游戏配置、厂家管理、点数配置、外部游戏测试、维护开关
- 体育游戏异常、上下分订单、厂商回调
- 客户端关注列表展示、配置表单、测试入口、维护标识
- 账服关注接口对接、点数同步、上下分事务、异常记录

### 运营管理

- 运营活动配置、公告发布、广播发送、多语言文本管理
- 站点地图、跨服中台、机器人客服、负责任系统
- 客户端关注活动展示、公告列表、多语言切换
- 账服关注活动规则、推送队列、多语言缓存、运营商同步

### 全民代管理

- 全民代列表、设置配置、分享好礼、佣金查询
- 团队查询、ROI 分析、排行榜、下级充值佣金
- 客户端关注列表筛选、团队树、数据图表、佣金详情
- 账服关注佣金计算、团队关系、ROI 统计、金币转换

### 合伙人与代理

- 合伙人管理、渠道配置、代理列表、佣金计算
- 佣金转账户、佣金记录查询
- 客户端关注列表、详情页、转账弹窗、渠道配置
- 账服关注佣金计算、渠道关联、转账事务、汇总统计

### 渠道管理

- 渠道列表、类型配置、数据查询、汇总统计
- Telegram 渠道、新玩家跟踪
- 客户端关注列表、详情页、数据图表
- 账服关注数据聚合、归属关联、Telegram 接口

### 厂商活动记录

- FC/FG/TADA/YELLOWBAT 活动数据同步
- 锦标赛、免费旋转、抽红包、礼包码兑换
- 客户端关注列表、详情、筛选、导出
- 账服关注数据同步、奖励发放、状态更新、厂商回调

### 财务系统

- 充值记录、提现记录、代付订单重置、人工金币修正
- 出款设置、人工转账、通道设置、代付银行配置
- 客户端关注列表页、审核按钮、配置表单、筛选条件
- 账服关注充值回调、提现审核、代付通道调用、金币修正、转账事务

### 玩家管理

- 玩家列表、机器人列表、在线列表、玩家分层
- 会员分组、标签管理、VIP 等级、回归玩家、账号注销
- 客户端关注列表页、详情页、编辑弹窗、筛选条件、批量操作
- 账服关注玩家查询、信息更新、分层计算、VIP 变更、标签关联

### 游戏及金流记录

- 游戏记录查询、用户金币变化记录、用户充提记录汇总
- 客户端关注记录列表、筛选条件、详情展示、导出按钮
- 账服关注记录查询接口、金币流水、聚合统计、数据同步
