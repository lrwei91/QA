# 运营活动领域

## 1. 覆盖模块

- `task-system` - 任务系统（日常任务、成长任务、活动任务）
- `tournament` - 锦标赛（FG/YELLOWBAT 等厂商锦标赛）
- `recharge-activity` - 充值活动（充值返利、充值抽奖）
- `cashback-activity` - 洗码活动（洗码返利、流水返利）
- `marketing-activity` - 营销活动（领券中心、签到、转盘）
- `vendor-activities` - 厂商活动记录（FC 抽红包、FG 免费旋转等）

## 2. 常见业务对象

### 2.1 任务系统
- 任务类型：日常任务、成长任务、一次性任务、限时任务
- 任务条件：投注任务、充值任务、连赢任务、连输任务
- 任务奖励：金币、道具、优惠券、抽奖机会
- 任务状态：进行中、已完成、已领取、已过期
- 任务刷新：每日刷新、每周刷新、手动刷新

### 2.2 锦标赛
- 锦标赛类型：按游戏厂家（FG/TADA/YELLOWBAT/FC）、按时间（日/周/月）
- 报名方式：免费报名、付费报名、自动报名、邀请制
- 排名规则：按赢利、按投注额、按连赢、按特定条件
- 奖励池：总奖金、名次分配、奖励类型（金币/道具/实物）
- 结算方式：自动结算、手动结算、定时结算
- 状态流转：未开始→进行中→已结束→已结算

### 2.3 充值活动
- 活动类型：首充活动、每日充值、累计充值、阶梯返利
- 充值档位：单笔满额、累计满额、多档位选择
- 返利类型：即时返利、手动领取、定时发放
- 返利形式：金币、优惠券、抽奖机会、道具
- 限制条件：每日次数限制、总参与次数、玩家资格限制

### 2.4 洗码活动
- 洗码定义：有效投注、有效输赢、流水计算
- 洗码比例：不同游戏类型不同比例（电子/真人/体育）
- 结算周期：每日结算、每周结算、每月结算
- 返利形式：金币返利、优惠券、升级奖励
- 风控规则：异常投注检测、对刷检测、限制条件

### 2.5 营销活动
- 领券中心：优惠券领取、使用条件、有效期
- 签到系统：连续签到、累计签到、补签机制
- 幸运转盘：抽奖次数、奖品池、保底机制
- 礼包系统：新手礼包、等级礼包、节日礼包
- 兑换商店：金币兑换、道具兑换、实物兑换

## 3. 高风险测试点

### 3.1 客户端

#### 任务系统
- 任务列表展示、进度条、完成状态标识
- 任务详情弹窗、条件说明、奖励预览
- 奖励领取按钮、领取成功提示、自动刷新
- 任务筛选（全部/进行中/已完成）、排序
- 任务刷新倒计时、手动刷新按钮

#### 锦标赛
- 锦标赛列表、详情展示、排名实时刷新
- 报名按钮状态（未开始/进行中/已满）
- 我的排名、我的成绩展示
- 奖励展示、名次对应奖励
- 倒计时展示、状态标签

#### 充值活动
- 活动页面展示、档位选择、进度展示
- 返利领取按钮、领取记录
- 活动规则展示、限制条件说明
- 充值金额输入、档位自动匹配

#### 洗码活动
- 洗码数据展示（有效投注、洗码值）
- 洗码进度条、返利计算器
- 返利记录、结算倒计时
- 不同游戏类型洗码比例说明

### 3.2 账服

#### 任务系统
- 任务条件判定逻辑（投注额/连赢/连输/充值）
- 任务进度更新、状态流转
- 奖励发放、金币入账、道具发放
- 任务刷新逻辑、过期清理
- 多任务并行处理、并发奖励发放

#### 锦标赛
- 报名逻辑校验（资格/余额/次数）
- 成绩同步与排名计算
- 定时结算、奖励批量发放
- 厂商数据回调处理
- 并列排名处理、破规则
- 结算后状态不可篡改

#### 充值活动
- 充值金额校验、档位匹配
- 返利资格判定（首充/每日/累计）
- 返利发放、次数限制校验
- 活动配置（时间/档位/奖励）
- 与其他活动互斥逻辑

#### 洗码活动
- 洗码计算逻辑（不同游戏类型比例）
- 有效投注判定（排除对刷/异常）
- 定时结算、批量返利
- 风控规则触发（异常投注冻结）
- 洗码数据修正、手动调整

## 5. 集成点

- **finance-system**：充值活动依赖充值记录、洗码活动依赖金币账目
- **player-management**：玩家资格判定、玩家分层参与条件
- **game-management**：游戏数据同步、投注记录获取
- **report-system**：活动数据报表、参与统计
- **vendor-activities**：厂商活动数据对接

## 6. 风险等级

| 风险等级 | 场景 | 说明 |
|----------|------|------|
| P0 | 奖励发放错误 | 多发/少发/错发，直接资金损失 |
| P0 | 排名计算错误 | 锦标赛排名错误，影响公平性 |
| P1 | 资格判定错误 | 不符合条件的玩家参与获利 |
| P1 | 数据不同步 | 前端进度与后端数据不一致 |
| P2 | 文案错误 | 活动规则展示错误，用户投诉风险 |
| P2 | 并发问题 | 多人同时领取奖励的幂等性 |
