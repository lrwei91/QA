# 全民代领域

## 1. 覆盖模块

- `affiliate-management` - 全民代管理（代理列表、佣金管理、团队管理）
- `partner-agent` - 合伙人与代理（合伙人渠道、代理佣金）
- `commission-system` - 佣金系统（佣金计算、佣金结算、佣金提现）
- `referral-system` - 推荐系统（推荐码、推荐关系、邀请奖励）

## 2. 常见业务对象

### 2.1 代理体系
- 代理等级：初级代理、中级代理、高级代理、合伙人
- 升级条件：下级人数、团队投注额、团队赢利、活跃度
- 代理权限：团队管理、佣金查看、数据报表、推广工具
- 代理状态：正常、冻结、注销、降级
- 代理类型：个人代理、团队代理、渠道代理

### 2.2 团队层级
- 团队结构：直属下级、间接下级、团队层级（一代/二代/.../N 代）
- 团队人数：直属人数、间接人数、总人数
- 团队业绩：团队投注额、团队赢利、团队活跃玩家数
- 层级关系：绑定关系、解绑、转移、继承
- 团队筛选：活跃代理、高产出代理、潜力代理

### 2.3 佣金体系
- 佣金类型：下级充值佣金、下级投注佣金、下级赢利佣金、团队奖金
- 佣金比例：固定比例、阶梯比例、等级比例、业绩比例
- 佣金结算：日结、周结、月结、即时结算
- 佣金提现：佣金转金币、佣金提现、佣金消费
- 佣金记录：佣金明细、结算记录、提现记录
- 佣金调整：手动调佣、活动加成、处罚扣减

### 2.4 推荐系统
- 推荐码：个人推荐码、专属链接、二维码
- 推荐关系：推荐人绑定、推荐时间、推荐渠道
- 推荐奖励：首充奖励、持续奖励、一次性奖励
- 推荐统计：推荐人数、推荐充值、推荐产出
- 推荐活动：推荐排行榜、推荐竞赛、额外奖励

### 2.5 ROI 分析
- ROI 计算：投入产出比、利润率、回本周期
- 成本项：佣金支出、活动成本、运营成本
- 收益项：下级投注额、下级赢利、充值金额
- 趋势分析：日 ROI、周 ROI、月 ROI、趋势图
- 代理画像：高价值代理、潜力代理、风险代理

### 2.6 排行榜
- 排行榜类型：佣金排行榜、推荐人数排行榜、ROI 排行榜、团队业绩排行榜
- 排行榜周期：日榜、周榜、月榜、总榜
- 排行榜奖励：排名奖励、额外佣金、荣誉称号
- 排行榜刷新：实时更新、定时刷新、缓存策略

## 3. 高风险测试点

### 3.1 客户端

#### 代理中心
- 代理中心入口、代理状态展示
- 团队数据概览（人数/业绩/佣金）
- 下级列表、直属/间接筛选
- 佣金明细、结算记录、提现记录
- ROI 数据展示、趋势图表
- 排行榜展示、我的排名

#### 团队管理
- 下级详情页、个人数据展示
- 团队层级展示、层级切换
- 代理数据筛选（活跃/产出/时间）
- 导出功能（下级列表/佣金记录）

#### 推广工具
- 推荐码生成、复制链接、二维码下载
- 推广素材（图片/文案/落地页）
- 推广链接追踪、来源统计

#### 佣金操作
- 佣金提现申请、提现账户绑定
- 佣金转金币、手续费展示
- 佣金明细查询、时间筛选
- 结算倒计时、预计结算时间

### 3.2 账服

#### 代理等级管理
- 升级条件判定（人数/业绩/活跃度）
- 降级条件判定（活跃度不足/业绩不达标）
- 等级变更通知、权益同步
- 冻结/解冻/注销处理

#### 团队关系管理
- 推荐关系绑定、唯一性校验
- 解绑条件、转移规则、继承逻辑
- 团队层级计算、路径缓存
- 批量绑定处理、并发校验

#### 佣金计算
- 充值佣金计算（比例/阶梯/等级）
- 投注佣金计算（有效投注/洗码比例）
- 赢利佣金计算（净赢利/净输赢）
- 团队奖金计算（条件达成/分配规则）
- 多层级佣金（一代/二代/.../N 代）
- 佣金结算、批量入账

#### 佣金提现
- 提现条件校验（最低限额/账户绑定）
- 手续费计算、实际到账金额
- 审核流程、自动/人工审核
- 打款处理、失败退回
- 佣金转金币、汇率计算

#### ROI 统计
- 成本聚合（佣金/活动/运营）
- 收益聚合（投注/赢利/充值）
- ROI 计算、趋势分析
- 代理画像、风险识别

#### 排行榜处理
- 数据聚合、排名计算
- 并列处理、破规则
- 定时刷新、缓存更新
- 奖励发放、排名变动通知

## 5. 集成点

- **player-management**：玩家代理身份、玩家数据同步
- **finance-system**：佣金账目、提现处理、充值记录
- **marketing-activities**：代理活动、推荐竞赛
- **report-system**：代理报表、团队数据统计
- **risk-management**：异常代理检测、刷量识别
- **game-golden-flow**：游戏记录、投注数据获取

## 6. 风险等级

| 风险等级 | 场景 | 说明 |
|----------|------|------|
| P0 | 佣金计算错误 | 比例错误/重复计算，直接资金损失 |
| P0 | 团队关系错误 | 绑定错误导致佣金归属错误 |
| P0 | 重复结算 | 同一业绩多次结算佣金 |
| P1 | 层级计算错误 | 间接下级层级计算错误 |
| P1 | 升级条件判定错误 | 不符合条件升级/符合条件未升级 |
| P1 | ROI 统计错误 | 成本/收益聚合错误 |
| P2 | 排行榜数据延迟 | 排名更新不及时 |
| P2 | 并发问题 | 同时绑定的关系冲突 |

## 7. 反作弊要求

- **刷量检测**：异常投注检测、对刷识别、机器人代理
- **关联账户**：同 IP 代理、同设备代理、资金关联
- **异常佣金**：超高佣金预警、异常增长预警
- **黑名单机制**：作弊代理拉黑、佣金冻结
