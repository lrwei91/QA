## 4. Common regression expansion rules

- 新增或修改业务流程 -> 补查 `operation-log` 与可能的 `notification-center`
- 修改统计口径 -> 补查指标来源模块与 `role-permission`
- 消息模板或路由变更 -> 补查触发源模块、失败重试和日志
- 报表数据源变化 -> 补查数据来源模块和 `player-management`、`finance-system`
- 风控规则变化 -> 补查 `finance-system`、`game-management`、`player-management`
- 系统配置变化 -> 补查 `role-permission`、`admin-user`、`operation-log`
- 短信/邮件渠道变化 -> 补查 `notification-center`、`player-management`
- 客服工单流程变化 -> 补查 `role-permission`、`notification-center`

## 5. Typical case ideas

### 运营看板

- 指标展示、筛选统计、时间边界、分组织口径、权限差异
- 客户端关注页面刷新、图表展示、筛选联动
- 账服关注统计查询、口径计算、缓存和数据一致性

### 操作日志

- 写日志、查日志、按模块筛选、按操作人筛选、审批轨迹回放
- 客户端关注查询页面、字段展示、脱敏展示
- 账服关注日志字段完整性、查询条件处理、异常补偿

### 消息中心

- 模板配置、发送、失败重试、收件人路由、发送记录
- 客户端关注配置页、记录页、按钮和状态展示
- 账服关注消息发送链路、渲染、回执、重试幂等、日志

### 报表系统

- 整体数据概览、游戏平台总览、全民代报表、玩家报表
- 支付平台报表、代付平台报表、用户留存报表、游戏留存报表
- 奖金任务报表、赠送金币报表、即时数据报表、收益报表
- 客户端关注报表列表、筛选条件、导出按钮、图表展示
- 账服关注数据聚合、口径计算、导出任务、权限过滤

### 风险管理

- 风控玩家管理、风控 CPF 管理、单异常管理、群异常管理
- 指定牌型配置、风险警示、警报系统、IP 区域配置、黑白名单
- 客户端关注配置页、警报展示、黑名单管理
- 账服关注风控规则校验、异常检测、警报触发、拦截逻辑

### 配置管理

- 系统配置管理、参数配置
- 客户端关注配置列表、编辑弹窗
- 账服关注配置读取、配置更新、缓存刷新

### 客服后台

- 工单信息管理、FAQ 配置、字段配置、客服列表、客服权限
- 客服操作统计、操作日志、服务 API 接入
- 客户端关注工单列表、FAQ 编辑、权限配置
- 账服关注工单分配、API 对接、状态更新

### 短信营销中心

- 短信渠道管理、短信模板配置、发送任务管理、发送记录查询
- 客户端关注渠道列表、模板编辑、任务创建
- 账服关注短信接口、发送队列、状态回执、失败重试

### 邮件管理

- 邮件渠道管理、邮件模板配置、发送任务管理、发送记录查询
- 邮箱验证、邮箱列表管理
- 客户端关注渠道列表、模板编辑、邮箱验证页
- 账服关注邮件接口、发送队列、状态回执、验证逻辑

### 系统设置

- 用户管理、权限管理、菜单显隐配置、菜单管理
