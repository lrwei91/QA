# 认证内容领域

## 1. 覆盖模块

- KYC 认证
- 人脸识别
- CPF 验证
- 税收报表
- 认证记录
- 登录安装（登录方式、安装包管理）

## 2. 常见业务对象

### 2.1 KYC 认证
- 认证类型：实名认证、手机认证、邮箱认证
- 认证状态：未认证、认证中、已通过、已拒绝
- 认证材料：身份证件、自拍照片、手持证件
- 认证信息：姓名、CPF、出生日期、地址

### 2.2 人脸识别
- 识别流程：活体检测、人脸比对、结果返回
- 识别状态：待识别、识别中、成功、失败
- 识别记录：识别时间、识别结果、相似度
- 重试机制：重试次数、重试间隔、失败锁定

### 2.3 CPF 验证
- CPF 格式：巴西税号格式校验
- CPF 有效性：官方接口验证
- CPF 唯一性：玩家 CPF 唯一性校验
- CPF 修改：修改限制、审核流程

### 2.4 税收报表
- 报表类型：大额投注报表、大额奖金报表、个人所得税报表
- 报表内容：玩家信息、投注/奖金金额、税率、应缴税额
- 报表周期：日报表、周报表、月报表
- 报表发送：发送记录、发送状态、接收确认

### 2.5 认证记录
- 记录内容：认证类型、认证时间、认证结果、审核人
- 记录查询：按玩家、按时间、按类型筛选
- 记录导出：数据导出、报表生成
- 记录审计：操作留痕、可追溯

## 3. 高风险测试点

### 3.1 客户端

#### 认证页面
- 认证引导文案、步骤说明
- 表单输入、字段校验
- 证件上传、格式校验、大小限制
- 识别结果展示、失败提示

#### 认证列表页
- 认证状态展示、筛选条件
- 认证记录详情、时间线
- 报表列表、查看/导出按钮

### 3.2 账服

#### KYC 认证处理
- 认证提交、数据校验
- 证件审核、人工审核流程
- 认证结果通知、状态更新
- 认证记录写入、审计日志

#### 人脸识别处理
- 活体检测、照片/视频上传
- 人脸比对、第三方接口调用
- 识别结果处理、状态同步
- 失败重试、锁定机制

#### CPF 验证处理
- CPF 格式校验、有效性验证
- CPF 唯一性校验、重复检测
- CPF 修改审核、记录留痕

#### 税收报表处理
- 报表数据聚合、税额计算
- 报表生成、定时任务
- 报表发送、发送记录
- 报表导出、数据格式

## 5. 集成点

- **玩家管理**：玩家认证状态、玩家详情
- **财务系统**：提现前认证校验、税收扣缴
- **风险管理**：异常认证检测、黑名单校验
- **报表系统**：认证统计报表、税收报表
- **游戏及金流记录**：大额投注/奖金触发报表

## 6. 风险等级

| 风险等级 | 场景 | 说明 |
|----------|------|------|
| P0 | 认证绕过 | 未完成认证允许提现，合规风险 |
| P0 | CPF 验证失效 | 虚假 CPF 通过验证，税务风险 |
| P1 | 识别阈值过低 | 非本人通过识别，账户安全风险 |
| P1 | 报表数据错误 | 税收报表数据错误，合规风险 |
| P2 | 认证状态不同步 | 前端显示与后端状态不一致 |
| P2 | 上传失败 | 证件上传失败、无明确提示 |

## 7. 合规要求

- **实名认证**：提现前必须完成实名认证
- **数据保护**：敏感信息加密存储、脱敏展示
- **审计留痕**：认证操作全程留痕、可追溯
- **报表合规**：税收报表按时生成、准确报送
