# 厂商活动记录领域

## 1. 覆盖模块

- FC 活动记录
- FG 活动记录
- 免费旋转记录
- 活动数据统计
- 净输赢查询

## 2. 常见业务对象

### 2.1 FC 活动记录
- 活动类型：FC 抽红包、FC 免费旋转、FC 转盘
- 参与记录：玩家 ID、活动 ID、参与时间、结果
- 奖励记录：奖励类型、奖励金额、发放状态
- 活动状态：进行中、已结束、已结算

### 2.2 FG 活动记录
- 活动类型：FG 免费旋转、FG 锦标赛、FG 返利
- 参与记录：玩家 ID、活动 ID、投注额、赢利
- 奖励记录：免费旋转次数、奖励金额
- 活动状态：报名中、进行中、已结束

### 2.3 免费旋转记录
- 旋转次数：剩余次数、已用次数、赠送次数
- 旋转结果：每次旋转结果、中奖金额
- 旋转状态：待旋转、旋转中、已完成
- 旋转期限：有效期、过期时间

### 2.4 活动数据统计
- 参与人数：总参与人数、新增参与人数
- 投注统计：总投注额、人均投注
- 输赢统计：总赢利、总输赢、净输赢
- 奖励统计：总奖励金额、人均奖励、ROI

### 2.5 净输赢查询
- 查询条件：玩家 ID、活动 ID、时间范围
- 统计维度：按玩家、按活动、按时间
- 统计指标：投注额、赢利、输赢、净输赢
- 查询限制：31 天查询限制、数据范围

## 5. 集成点

- **游戏及金流记录**：游戏数据同步、投注记录
- **财务系统**：奖励发放、账务处理
- **玩家管理**：玩家活动参与记录
- **报表系统**：活动统计报表、ROI 分析
- **游戏管理**：游戏厂家配置、活动配置

## 6. 风险等级

| 风险等级 | 场景 | 说明 |
|----------|------|------|
| P0 | 奖励发放错误 | 多发/少发奖励，资金损失 |
| P0 | 旋转次数错误 | 次数计算错误，引发投诉 |
| P1 | 数据同步延迟 | 厂家数据同步不及时 |
| P1 | 统计口径错误 | 净输赢计算错误 |
| P2 | 记录查询失败 | 大数据量查询超时 |
| P2 | 过期清理失效 | 过期旋转次数未清理 |

## 7. 业务要求

- **数据准确**：活动记录、统计数据准确
- **发放及时**：奖励及时发放、不遗漏
- **查询高效**：大数据量查询性能优化
- **对账机制**：与厂家定期对账、差异处理
