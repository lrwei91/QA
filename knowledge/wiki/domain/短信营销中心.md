# 短信营销中心领域

## 1. 覆盖模块

- 短信模板
- 群发配置
- 发送记录
- 效果分析
- 接收管理
- 验证码通知（验证码发送、短信通知、频率限制）

## 2. 常见业务对象

### 2.1 短信模板
- 模板信息：模板 ID、模板名称、模板内容、变量
- 模板类型：验证码模板、通知模板、营销模板
- 模板状态：草稿、审核中、已通过、已拒绝
- 模板版本：版本号、更新时间、更新人

### 2.2 群发配置
- 发送对象：全体玩家、筛选玩家、指定玩家
- 发送时间：立即发送、定时发送
- 发送策略：一次性发送、分批发送、频率控制
- 发送限制：发送时间限制、频次限制、黑名单

### 2.3 发送记录
- 记录信息：发送 ID、模板 ID、接收人、发送时间
- 发送状态：待发送、发送中、已发送、发送失败
- 短信内容：短信正文、变量替换后内容
- 回调数据：送达状态、阅读状态、点击链接

### 2.4 效果分析
- 发送统计：发送数量、成功数量、失败数量
- 送达统计：送达率、失败原因分析
- 转化统计：点击率、转化率、ROI
- 趋势分析：日发送量、周发送量、月发送量

### 2.5 接收管理
- 黑名单管理：拒收玩家列表、拉黑原因
- 退订管理：玩家退订记录、退订时间
- 频次控制：单玩家发送频次限制
- 投诉处理：投诉记录、处理结果

## 5. 集成点

- **玩家管理**：玩家手机号、玩家筛选
- **系统设置**：短信渠道配置、发送权限
- **风险管理**：发送频次限制、黑名单管理
- **报表系统**：发送统计报表、效果分析
- **操作日志**：发送操作留痕、审计轨迹

## 6. 风险等级

| 风险等级 | 场景 | 说明 |
|----------|------|------|
| P0 | 短信误发 | 错误内容群发，品牌影响 |
| P0 | 敏感内容发送 | 违规内容未审核通过 |
| P1 | 发送频率失控 | 短时间大量发送，被运营商封禁 |
| P1 | 黑名单失效 | 拒收玩家仍收到短信 |
| P2 | 发送延迟 | 定时发送未按时执行 |
| P2 | 统计误差 | 发送数据统计不准确 |

## 7. 合规要求

- **内容审核**：营销短信内容需审核
- **用户同意**：仅向同意接收的玩家发送
- **退订机制**：提供便捷退订方式
- **频次控制**：避免过度打扰玩家
