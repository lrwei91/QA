# 游戏管理领域

## 1. 覆盖模块

- 游戏列表
- 游戏配置
- 游戏上下架
- 游戏分类
- 游戏厂家管理
- 自定义厂家（自定义游戏厂家配置）
- 大厅游戏入口（大厅入口配置、游戏入口展示）
- 冷热排行榜（热门游戏、冷门游戏推荐）

## 2. 常见业务对象

### 2.1 游戏列表
- 游戏信息：游戏 ID、游戏名称、游戏类型、游戏厂家
- 游戏状态：上架、下架、维护、停用
- 游戏配置：投注范围、最大派奖、 RTP 配置
- 游戏标签：热门、新品、推荐、独家

### 2.2 游戏配置
- 投注配置：最小投注、最大投注、投注档位
- 派奖配置：最大派奖、派奖倍数、奖池配置
- 游戏参数：游戏速度、音效开关、快捷模式
- 风控配置：最大输赢限制、异常检测阈值

### 2.3 游戏上下架
- 上架流程：申请、审核、配置、发布
- 下架流程：申请、审核、清场、下线
- 上下架时间：定时上下架、立即生效
- 上下架范围：全平台、部分渠道、部分玩家

### 2.4 游戏分类
- 分类类型：电子、真人、体育、棋牌
- 分类管理：分类新增、编辑、排序、停用
- 游戏关联：游戏绑定分类、多分类支持
- 分类展示：前台展示配置、排序规则

### 2.5 游戏厂家管理
- 厂家信息：厂家 ID、厂家名称、对接方式
- 厂家配置：API 配置、回调配置、结算配置
- 厂家状态：正常、维护、停止合作
- 厂家数据：游戏数量、投注额、输赢统计

## 3. 高风险测试点

### 3.1 客户端

#### 游戏列表页
- 游戏列表展示、分类筛选
- 游戏搜索、模糊匹配
- 游戏标签展示（热门/新品）
- 游戏状态标识（上架/下架/维护）

#### 游戏配置页
- 配置表单展示、字段校验
- 配置保存、取消操作
- 配置历史记录、版本对比
- 配置生效提示

### 3.2 账服

#### 游戏管理
- 游戏新增/编辑、数据校验
- 游戏上下架处理、状态同步
- 游戏配置更新、缓存刷新
- 厂家对接配置、联调测试

#### 游戏数据同步
- 游戏厂家数据拉取、定时任务
- 游戏数据更新、差异处理
- 游戏状态监控、异常告警

#### 配置生效
- 配置审核流程、权限控制
- 配置发布、灰度发布
- 配置回滚、紧急下线

## 5. 集成点

- **游戏及金流记录**：游戏数据同步、投注记录
- **财务系统**：游戏结算、账务处理
- **玩家管理**：玩家游戏权限、游戏记录
- **风险管理**：游戏风控、异常检测
- **报表系统**：游戏统计报表、厂家报表

## 6. 风险等级

| 风险等级 | 场景 | 说明 |
|----------|------|------|
| P0 | 配置错误 | 投注/派奖配置错误，资金损失 |
| P0 | 上下架失效 | 应下架游戏仍可访问 |
| P1 | 数据不同步 | 游戏数据与厂家不一致 |
| P1 | 缓存未刷新 | 配置变更后未生效 |
| P2 | 分类错误 | 游戏分类展示错误 |
| P2 | 搜索失效 | 游戏搜索功能异常 |

## 7. 运维要求

- **灰度发布**：新游戏/配置变更支持灰度
- **紧急下线**：问题游戏可快速下线
- **监控告警**：游戏异常自动告警
- **数据备份**：配置变更有备份可回滚
