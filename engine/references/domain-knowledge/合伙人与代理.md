# 合伙人与代理领域

## 1. 覆盖模块

- 合伙人管理
- 代理管理
- 佣金结算
- 团队管理
- 推广数据
- 代理风控（代理风险识别、代理风控规则）

## 2. 常见业务对象

### 2.1 合伙人管理
- 合伙人信息：合伙人 ID、姓名、联系方式、加入时间
- 合伙人等级：初级合伙人、中级合伙人、高级合伙人
- 合伙人权益：分成比例、专属奖励、技术支持
- 合伙人状态：正常、冻结、终止合作

### 2.2 代理管理
- 代理信息：代理 ID、代理名称、推荐码、联系方式
- 代理等级：一级代理、二级代理、三级代理
- 代理权益：佣金比例、团队分成、推广支持
- 代理状态：正常、冻结、注销

### 2.3 佣金结算
- 佣金类型：推荐佣金、团队佣金、业绩佣金
- 结算周期：日结、周结、月结
- 结算状态：待结算、已结算、已发放
- 结算记录：结算单、发放记录、明细

### 2.4 团队管理
- 团队结构：直属团队、间接团队、团队层级
- 团队成员：团队成员列表、成员状态
- 团队业绩：团队投注额、团队赢利、团队人数
- 团队分成：分层级分成比例、分成计算

### 2.5 推广数据
- 推荐数据：推荐人数、推荐充值、推荐活跃
- 业绩数据：个人业绩、团队业绩、ROI
- 佣金数据：佣金收入、佣金提现、佣金记录
- 趋势数据：日数据、周数据、月数据

## 5. 集成点

- **玩家管理**：代理人玩家身份、玩家数据
- **财务系统**：佣金账目、提现处理
- **运营活动**：代理活动、推广竞赛
- **报表系统**：代理报表、团队数据统计
- **风险管理**：异常代理检测、刷量识别
- **游戏及金流记录**：团队游戏数据、投注记录

## 6. 风险等级

| 风险等级 | 场景 | 说明 |
|----------|------|------|
| P0 | 佣金计算错误 | 比例错误/重复计算，资金损失 |
| P0 | 团队关系错误 | 绑定错误导致佣金归属错误 |
| P0 | 重复结算 | 同一业绩多次结算佣金 |
| P1 | 层级计算错误 | 间接团队层级计算错误 |
| P1 | 等级变更失效 | 符合条件未升级/不应降级被降级 |
| P2 | 数据不同步 | 推广数据更新不及时 |

## 7. 业务要求

- **结算准确**：佣金计算准确、不遗漏
- **发放及时**：按时发放佣金、不拖欠
- **数据透明**：代理数据实时可查
- **风险可控**：异常代理检测、防刷机制
