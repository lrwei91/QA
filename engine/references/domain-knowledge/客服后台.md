# 客户后台领域

## 1. 覆盖模块

- 客户列表
- 客服工单
- 客户备注
- 客户标签
- 客户详情

## 2. 常见业务对象

### 2.1 客户列表
- 客户信息：客户 ID、昵称、手机号、注册时间
- 客户状态：正常、冻结、注销、黑名单
- 客户等级：VIP 等级、经验值、成长值
- 资产信息：金币余额、充值总额、提现总额
- 筛选条件：注册时间、最后登录、充值金额、输赢情况

### 2.2 客服工单
- 工单信息：工单 ID、工单标题、工单内容、提交时间
- 工单类型：充值问题、提现问题、游戏问题、账号问题、投诉建议
- 工单状态：待处理、处理中、已解决、已关闭
- 工单优先级：普通、紧急、重要
- 工单分配：处理人、分配时间、处理时长

### 2.3 客户备注
- 备注信息：备注 ID、备注内容、备注人、备注时间
- 备注类型：普通备注、风险备注、重要备注
- 备注可见性：个人可见、团队可见
- 备注历史：历史备注记录、修改记录

### 2.4 客户标签
- 标签类型：充值标签、活跃度标签、风险标签、VIP 标签
- 标签管理：添加标签、移除标签、批量标签
- 标签筛选：按标签分组、标签组合筛选
- 标签统计：各标签玩家数量、标签分布

### 2.5 客户详情
- 基本信息：昵称、手机号、CPF、邮箱
- 账户信息：余额、充值记录、提现记录
- 游戏记录：投注记录、输赢统计、游戏偏好
- 活动记录：参与的活动、领取的奖励
- 操作日志：账号变更、备注记录、工单记录

## 3. 高风险测试点

### 3.1 客户端

#### 客户列表页
- 客户列表展示、分页、排序
- 筛选条件栏、搜索框
- 客户状态标识、VIP 标签
- 批量操作按钮、导出按钮

#### 客服工单页
- 工单列表展示、筛选
- 工单详情查看、回复功能
- 工单分配、转交处理
- 工单状态变更、关闭工单

#### 客户详情页
- 客户基本信息展示
- 资产信息展示（余额、充值、提现）
- 游戏记录列表、筛选
- 操作按钮（冻结/备注/标签）

### 3.2 账服

#### 客户查询
- 客户列表查询接口、分页处理
- 筛选条件处理、排序逻辑
- 客户详情数据聚合

#### 客服工单处理
- 工单创建、数据校验
- 工单分配、处理人变更
- 工单回复、通知推送
- 工单关闭、满意度评价

#### 客户操作
- 账号冻结/解冻处理、权限校验
- 备注添加、标签管理
- 操作日志写入、审计轨迹

#### 数据一致性
- 客户余额准确性校验
- 操作事务处理、回滚机制
- 缓存刷新策略

## 5. 集成点

- **玩家管理**：玩家数据同步、玩家详情
- **财务系统**：客户充值/提现记录
- **游戏及金流记录**：客户游戏记录
- **风险管理**：风险客户标识、黑名单管理
- **认证内容**：客户认证状态
- **操作日志**：操作留痕、审计轨迹

## 6. 风险等级

| 风险等级 | 场景 | 说明 |
|----------|------|------|
| P0 | 账号冻结失效 | 风险账号未及时冻结，资金损失 |
| P0 | 余额数据错误 | 客户余额显示错误，引发投诉 |
| P1 | 权限校验漏洞 | 越权操作客户数据 |
| P1 | 工单处理延迟 | 客户工单未及时处理，满意度下降 |
| P2 | 数据不同步 | 前端展示与后端数据不一致 |
| P2 | 备注泄露 | 内部备注被客户看到 |

## 7. 服务要求

- **响应及时**：客户工单及时响应、处理
- **数据准确**：客户数据准确无误
- **隐私保护**：客户敏感信息脱敏展示、加密存储
- **操作审计**：关键操作留痕、可追溯
