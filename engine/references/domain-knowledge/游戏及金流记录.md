# 游戏及金流记录领域

## 1. 覆盖模块

- 游戏记录
- 投注记录
- 资金流水
- 派奖记录
- 游戏统计

## 2. 常见业务对象

### 2.1 游戏记录
- 游戏对局：对局 ID、游戏类型、游戏厂家、开始时间、结束时间
- 参与玩家：玩家 ID、昵称、头像
- 游戏结果：输赢金额、投注额、派奖金额
- 游戏状态：进行中、已完成、异常终止

### 2.2 投注记录
- 投注信息：投注 ID、游戏 ID、投注金额、投注线数
- 投注结果：中奖金额、中奖类型、中奖线
- 投注时间：投注时间、结算时间
- 投注状态：待结算、已结算、已取消

### 2.3 资金流水
- 流水类型：投注、派奖、充值、提现、活动奖励、补偿
- 流水金额：变动金额、变动前余额、变动后余额
- 流水时间：发生时间、入账时间
- 关联信息：关联订单 ID、关联游戏 ID、备注

### 2.4 派奖记录
- 派奖信息：派奖 ID、派奖金额、派奖类型
- 派奖条件：中奖条件、中奖倍数、中奖线
- 派奖状态：待派奖、派奖中、已完成、失败
- 派奖时间：派奖时间、到账时间

### 2.5 游戏统计
- 统计维度：按游戏、按时间、按玩家、按厂家
- 统计指标：投注额、输赢、派奖率、参与人数
- 统计周期：实时、日统计、周统计、月统计
- 统计报表：游戏报表、厂家报表、玩家报表

## 3. 高风险测试点

### 3.1 客户端

#### 游戏记录页
- 游戏记录列表展示、分页
- 筛选条件（游戏类型、时间范围、输赢情况）
- 游戏详情弹窗、对局回放
- 导出功能

#### 投注记录页
- 投注记录列表、分页
- 投注详情展示（投注额、中奖额、中奖类型）
- 筛选条件（时间、游戏、投注额范围）
- 统计汇总（总投注、总输赢）

### 3.2 账服

#### 游戏记录处理
- 游戏对局数据同步、厂家回调处理
- 游戏结果校验、反作弊检测
- 游戏记录写入、事务处理

#### 投注记录处理
- 投注数据接收、校验
- 投注结算、派奖计算
- 投注状态流转、异常处理

#### 资金流水处理
- 流水写入、事务一致性
- 余额更新、并发控制
- 流水查询、聚合统计

#### 数据统计
- 实时统计、定时任务
- 数据聚合、缓存策略
- 统计报表生成、导出

## 5. 集成点

- **游戏管理**：游戏配置、游戏状态同步
- **财务系统**：资金流水、余额变更
- **玩家管理**：玩家数据、玩家筛选
- **风险管理**：异常投注检测、反作弊
- **报表系统**：游戏统计报表、数据分析

## 6. 风险等级

| 风险等级 | 场景 | 说明 |
|----------|------|------|
| P0 | 投注记录丢失 | 玩家投注未记录，引发纠纷 |
| P0 | 派奖计算错误 | 派奖金额错误，资金损失 |
| P0 | 流水写入失败 | 资金流水缺失，账目不平 |
| P1 | 数据同步延迟 | 游戏记录更新不及时 |
| P1 | 结算异常 | 投注未结算或重复结算 |
| P2 | 统计误差 | 统计数据与实际有偏差 |

## 7. 数据一致性

- **事务处理**：投注、派奖、流水写入在同一事务
- **幂等性**：厂家回调多次处理，保证数据不重复
- **对账机制**：与游戏厂家定期对账，差异处理
