# 玩家管理领域

## 1. 覆盖模块

- 玩家列表
- 玩家详情
- 账号管理
- 玩家标签
- 玩家筛选
- VIP 管理（VIP 等级、VIP 权益、VIP 升级）
- CPF 管控（CPF 验证、CPF 限制）

## 2. 常见业务对象

### 2.1 玩家列表
- 玩家信息：玩家 ID、昵称、头像、注册时间、最后登录
- 玩家状态：正常、冻结、注销、黑名单
- 玩家等级：VIP 等级、经验值、成长值
- 资产信息：金币余额、绑定金币、优惠金额
- 筛选条件：注册时间、最后登录、充值金额、输赢情况

### 2.2 玩家详情
- 基本信息：昵称、手机号、CPF、邮箱
- 账户信息：余额、充值记录、提现记录
- 游戏记录：投注记录、输赢统计、游戏偏好
- 活动记录：参与的活动、领取的奖励
- 操作日志：账号变更、备注记录

### 2.3 玩家标签
- 标签类型：充值标签、活跃度标签、风险标签、VIP 标签
- 标签管理：添加标签、移除标签、批量标签
- 标签筛选：按标签分组、标签组合筛选

### 2.4 账号管理
- 账号操作：冻结账号、解冻账号、注销账号
- 密码管理：重置密码、绑定手机、绑定邮箱
- 备注管理：添加备注、查看历史备注
- 权限控制：操作权限、数据范围

## 3. 高风险测试点

### 3.1 客户端

#### 玩家列表页
- 玩家列表展示、分页、排序
- 筛选条件栏、搜索框
- 玩家状态标识（VIP 标签、状态标签）
- 批量操作按钮、导出按钮

#### 玩家详情页
- 玩家基本信息展示
- 资产信息展示（余额、充值、提现）
- 游戏记录列表、筛选
- 操作按钮（冻结/备注/标签）

### 3.2 账服

#### 玩家查询
- 玩家列表查询接口、分页处理
- 筛选条件处理、排序逻辑
- 玩家详情数据聚合

#### 玩家操作
- 账号冻结/解冻处理、权限校验
- 重置密码处理、通知发送
- 备注添加、标签管理
- 操作日志写入

#### 数据一致性
- 玩家余额准确性校验
- 操作事务处理、回滚机制
- 缓存刷新策略

## 5. 集成点

- **财务系统**：玩家余额、充值记录、提现记录
- **游戏及金流记录**：游戏记录、投注数据
- **运营活动**：活动参与记录、奖励发放
- **风险管理**：风险玩家标识、黑名单管理
- **认证内容**：KYC 认证状态、认证记录
- **操作日志**：操作留痕、审计轨迹

## 6. 风险等级

| 风险等级 | 场景 | 说明 |
|----------|------|------|
| P0 | 账号冻结失效 | 风险账号未及时冻结，资金损失 |
| P0 | 余额数据错误 | 玩家余额显示错误，引发投诉 |
| P1 | 权限校验漏洞 | 越权操作玩家数据 |
| P1 | 操作日志缺失 | 关键操作无记录，无法追溯 |
| P2 | 数据不同步 | 前端展示与后端数据不一致 |
| P2 | 并发问题 | 同时操作同一玩家的冲突处理 |

## 7. 数据合规

- **隐私保护**：敏感信息脱敏展示、加密存储
- **数据权限**：操作权限控制、数据范围限制
- **操作审计**：关键操作留痕、可追溯
