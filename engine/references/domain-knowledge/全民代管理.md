# 全民代管理领域

## 1. 覆盖模块

- 代理列表
- 佣金管理
- 团队管理
- 推广数据
- 排行榜

## 2. 常见业务对象

### 2.1 代理列表
- 代理信息：代理 ID、昵称、推荐码、加入时间
- 代理等级：初级代理、中级代理、高级代理
- 代理状态：正常、冻结、注销
- 代理数据：下级人数、团队业绩、佣金收入

### 2.2 佣金管理
- 佣金类型：推荐佣金、团队佣金、业绩佣金
- 佣金记录：佣金明细、结算记录、发放记录
- 佣金状态：待结算、已结算、已发放
- 佣金提现：提现申请、审核、打款

### 2.3 团队管理
- 团队结构：直属下级、间接下级、团队层级
- 团队成员：成员列表、成员状态、成员数据
- 团队业绩：团队投注额、团队赢利、活跃人数
- 团队分成：分层级分成、分成比例、分成计算

### 2.4 推广数据
- 推荐数据：推荐人数、推荐充值、推荐活跃
- 业绩数据：个人业绩、团队业绩、ROI
- 转化数据：注册转化率、充值转化率
- 趋势数据：日数据、周数据、月数据

### 2.5 排行榜
- 排行榜类型：佣金榜、推荐人数榜、ROI 榜、团队业绩榜
- 排行榜周期：日榜、周榜、月榜、总榜
- 排行榜奖励：排名奖励、额外佣金、荣誉
- 排行榜刷新：实时更新、定时刷新

## 5. 集成点

- **玩家管理**：代理人玩家身份、玩家数据
- **财务系统**：佣金账目、提现处理
- **运营活动**：代理活动、推广竞赛
- **报表系统**：代理报表、团队数据统计
- **风险管理**：异常代理检测、刷量识别
- **游戏及金流记录**：团队游戏数据、投注记录

## 6. 风险等级

| 风险等级 | 场景 | 说明 |
|----------|------|------|
| P0 | 佣金计算错误 | 比例错误/重复计算，资金损失 |
| P0 | 团队关系错误 | 绑定错误导致佣金归属错误 |
| P0 | 重复结算 | 同一业绩多次结算佣金 |
| P1 | 层级计算错误 | 间接团队层级计算错误 |
| P1 | 等级变更失效 | 符合条件未升级/不应降级被降级 |
| P2 | 排行榜数据延迟 | 排名更新不及时 |

## 7. 业务要求

- **结算准确**：佣金计算准确、不遗漏
- **发放及时**：按时发放佣金、不拖欠
- **数据透明**：代理数据实时可查
- **风险可控**：异常代理检测、防刷机制
